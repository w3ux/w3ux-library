# Ledger Connect

Ledger hardware wallet adaptor for @w3ux/react-connect-kit

## Installation

```bash
npm install @w3ux/ledger-connect
```

or

```bash
yarn add @w3ux/ledger-connect
```

or

```bash
pnpm add @w3ux/ledger-connect
```

## Usage

```typescript
import { /* your imports */ } from '@w3ux/ledger-connect'
```

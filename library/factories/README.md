# Factories

A collection of general purpose TypeScript factories

## Installation

```bash
npm install @w3ux/factories
```

or

```bash
yarn add @w3ux/factories
```

or

```bash
pnpm add @w3ux/factories
```

## Usage

```typescript
import { /* your imports */ } from '@w3ux/factories'
```

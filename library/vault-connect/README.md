# Vault Connect

Polkadot Vault (QR-based) wallet adaptor for @w3ux/react-connect-kit

## Installation

```bash
npm install @w3ux/vault-connect
```

or

```bash
yarn add @w3ux/vault-connect
```

or

```bash
pnpm add @w3ux/vault-connect
```

## Usage

```typescript
import { /* your imports */ } from '@w3ux/vault-connect'
```
